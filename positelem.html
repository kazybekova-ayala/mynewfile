Элементы с position:static
Элементы c position: static являются обычными элементами потока,
они располагаются в соответствии с нормальным потоком документа
Является значением по умолчанию для всех элементов HTML.
Элементы с position: absolute
Элементы с position: absolute позиционируются относительно ближайшего предка с position отличным от static.
Если таких предков нет, элемент позиционируется относительно начального контейнера (обычно ).
Элементы с position: relative
Они располагаются в обычном потоке документа,
но можно использовать свойства , right, bottom и left для их смещения относительно их нормального
Элементы с position: fixed
Элементы с position: sticky ведут себя как relative до тех пор, пока не достигнут заданной позиции при прокрутке, после чего ведут себя как fixed.
